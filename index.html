Diseño Web

HTML

HTML es un lenguaje que se usa para estructurar las paginas web el cual es un fundamento en el desarrollo web

CSS

CSS se usa para dar estilos a los elementos de la pagina web para que no se vea muy sencillo como un documento de word

Java Script

Java Script se usa para dar dinamismo y funciones a la pagina para que sea mas dinámica

Angular

Angular es un framework que se usa para crear paginas web de una mejor manera
